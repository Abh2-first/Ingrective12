# üçé Can Read and Eat

**Can Read and Eat** is a modern Next.js web app that helps users scan packaged food products, understand their ingredients in simple language, and evaluate how suitable they are for their health conditions. The app connects to a Hugging Face OCR model to extract information from food labels and presents a health score for easy understanding.

---

## üåü Features

- Multi-page responsive app using **Next.js** + **Tailwind CSS**
- Upload and scan product labels with AI-powered OCR via Hugging Face
- Displays ingredients and calculates a health score
- Informative pages:
  - Home / Intro
  - Scan product
  - User profile (future tracking)
  - Sample product examples
  - Harmful ingredients info
  - About & Contact
- Clean modern UI with gradient background and responsive design
